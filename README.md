# 🌸 सखीJunction  
**Empowering Women Through Community, Connection & AI-Driven Wellness**

Sakhi Junction is a full-stack platform built to support women's wellness through **community interaction, curated resources, meaningful connections, and collaboration**.  
Designed with a blend of **React.js, Node.js, MongoDB, Express.js, JWT**, and **Figma**, the platform integrates AI-powered assistance with health tracking and self-care tools.

🔗 **Live Demo:** [she-codes-one.vercel.app](https://she-codes-one.vercel.app/)

---

# 🌟 Features

| Module | Highlights |
|---|---|
| 🧠 **AI-Powered Support** | Real-time chatbot guidance · Personalized self-care advice · Curated wellness articles & blogs |
| 🌼 **Community & Connection** | Community feed with stories · Reactions & comments · Encrypted private chats via Socket.io |
| 🩺 **Health & Wellness** | Menstrual & wellness tracker with Recharts · Symptom logging with history · Smart pattern insights |
| 🗣️ **HerVoice – Self-Care** | Scheduled motivational messages · Multilingual audio affirmations (Web Speech API) · Daily journaling & check-ins |
| 🔐 **Authentication** | JWT-based login/signup · Protected routes · Role-based access control |
| 🌙 **Theming** | Light/Dark mode · Mobile-first responsive design · WCAG-aligned accessibility |

## 🚀 Getting Started

```bash
git clone https://github.com/drikshathakur786/HackWithHer.git
cd sakhijunction
npm install
npm run dev
```

> **Note:** Ensure MongoDB is running and configure `.env` variables for JWT, Cloudinary, and API keys. See `.env.example` for reference.
